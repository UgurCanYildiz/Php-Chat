
# PHP CHAT APP

PHP ve Mysql kullanarak kullanıcıların kendi aralarında mesajlaşabiliceği özel bir chat uygulamasıdır.Uygulama gizli bir chat uygulaması fikrinde olduğu için dışarıdan kayıt olunamaz aktif olan bir kullanıcı hesap oluşturabilir. Projemdeki dosyaları indirerek ve mysql ' de chat adında bir veri tabanı oluşturup projemdeki chat.sql dosyasını içeriye atarak aktifleştirebilirsiniz. 




## Kullanılan Teknolojiler

**İstemci:** PHP

**Sunucu:** MYSQL
